## Descrição
Site estilo catálogo de streaming (tema Netflix) para exibição de prévias e venda de conteúdo digital. O projeto foca em uma experiência visual premium, com animações, design responsivo e interatividade.

## Estrutura do Projeto
- **index.html**: Página principal com carrossel e categorias.
- **style.css**: Estilos globais, incluindo temas dark, animações e responsividade.
- **script.js**: Lógica do carrossel (Swiper), proteções anti-clone e players de vídeo.
- **SubPacks/**: Páginas individuais de prévia para cada coleção.
- **vercel.json**: Configuração de deploy e headers de segurança.

## Tecnologias Utilizadas
- **HTML5 Semantic**: Estrutura otimizada.
- **CSS3 Modern**: Flexbox, Grid, Animações, Variáveis CSS.
- **JavaScript (ES6+)**: Interatividade leve e manipulação de DOM.
- **Swiper.js**: Biblioteca para carrosséis touch.
- **Cloudinary**: Hospedagem otimizada de imagens e vídeos.

## Como Rodar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/Gumballxnz/site-hot.git
```

2. Abra o arquivo `index.html` no seu navegador ou use uma extensão como Live Server.

## Melhorias Recentes
- **Proteção Anti-Clone**: Bloqueio de download, clique direito e atalhos de cópia.
- **Performance Mobile**: Remoção de destaque azul (tap highlight) e otimização de toque.
- **Clean URLs**: Configuração para URLs limpas (sem .html) no Netlify/Vercel.
- **Imagens e Vídeos**: Todos os assets hospedados em CDN (Cloudinary) para carregamento rápido.
